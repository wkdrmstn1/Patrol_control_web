### 1. 핵심 관제 및 모니터링 (Core Dashboard)

- **`MainScreen.tsx`**
  - **역할**: 관제 시스템 통합 대시보드
  - **기능**: 실시간 카메라 스트리밍과 2D SLAM 지도를 한 화면에 제공합니다. 지도 위에 로봇의 현재 위치(Nav 아이콘)와 주행 경로가 실시간 렌더링되며, 자율주행(1회/무한반복), 수동 조작, 파노라마 촬영, 특정 구역 이동 등의 직접적인 하드웨어 제어 명령을 내릴 수 있습니다. 또한 객체 감지(화재/사람) 및 음성 인식 시스템 토글 스위치를 포함합니다.

- **`ManualScreen.tsx`**
  - **역할**: 원격 수동 조작 인터페이스
  - **기능**: 실시간 카메라 스트리밍, `W`, `A`, `S`, `D` 키보드 입력 이벤트를 감지하여 로봇의 전/후진 및 방향 제어 명령을 서버로 실시간 전송합니다. 스페이스바(Space) 또는 키보드에서 손을 뗄 경우(Key Up) 즉시 정지 명령을 전송하여 안전한 조작을 지원합니다.

- **`StatusBar.tsx`**
  - **역할**: 전역 최상단 상태 표시바
  - **기능**: 로봇의 통신 연결 상태, 배터리 잔량, 현재 위치 구역명, 객체 감지 상태 및 현재 시간을 실시간 표기합니다. 도난, 화재, 침입자 등 위험 상황 감지 시 텍스트 색상을 붉은색으로 점멸(애니메이션)시켜 관리자에게 즉각적인 시각적 경고를 제공합니다.

### 2. 경로 및 로그 관리 (Navigation & Logs)
- **`RouteSettingModal.tsx`**
  - **역할**: 자율주행 경로 및 방향 설정 도구
  - **기능**: 지도 위를 클릭해 시작점과 경유지(Waypoint) 마커를 생성합니다. 마커를 드래그해 위치를 미세 조정하고, 화살표를 회전시켜 로봇이 도착했을 때 바라볼 방향(Yaw)을 설정합니다. 각 지점의 A/B 구역 할당 및 주행 순서 변경 기능도 지원합니다.

- **`LogHistoryModal.tsx`**
  - **역할**: 이상 감지 이벤트 로그 리스트
  - **기능**: 화재, 침입자, 사람 감지 이벤트 및 파노라마 촬영 기록을 테이블 형태로 조회합니다. 현재 로그인한 사용자가 최고 관리자(`admin`)인 경우, 다중 선택 체크박스와 데이터 영구 삭제(휴지통) 버튼이 활성화됩니다.

- **`ImageModal.tsx`**
  - **역할**: 로그 사진 상세 보기 팝업
  - **기능**: `LogHistoryModal`에서 선택한 특정 이벤트의 고화질 원본 사진을 화면 중앙에 확대하여 렌더링합니다. 이미지 로드 실패 시 대체 이미지(Fallback)를 띄워 UI 깨짐을 방지합니다.

### 3. 상태 전용 스크린 (Status Screens)
- **`ChargingScreen.tsx`**
  - **역할**: 로봇 배터리 충전 상태 화면
  - **기능**: 현재 배터리 잔량(%)에 맞춰 충전 진행 바의 길이와 색상(적색 → 황색 → 녹색)이 동적으로 변하는 애니메이션 인터페이스를 제공합니다.

- **`StandbyScreen.tsx`**
  - **역할**: 로봇 임무 대기 화면
  - **기능**: 시스템 로그인 직후 또는 임무 대기 시 출력되는 랜딩 페이지로, 메인 관제 화면 진입 또는 충전 스테이션 이동의 분기점 역할을 합니다.

- **`PanoramaScreen.tsx`**
  - **역할**: 파노라마 촬영 결과 뷰어
  - **기능**: 파노라마 촬영 명령 하사 시 로딩 화면을 출력하며, 백엔드를 주기적으로 확인(Polling)해 병합 완료된 180도 파노라마 이미지가 등록되면 화면에 출력하고 촬영 위치 및 시간 정보를 제공합니다.

### 4. 보안 및 인증 (Authentication)
- **`LoginScreen.tsx`**
  - **역할**: 관리자 로그인 페이지
  - **기능**: 사원증 번호(ID)와 비밀번호를 통해 서버와 인증을 수행하며, 성공 시 상태 관리 컨텍스트(`AuthContext`)에 유저 정보를 등록하여 관제 시스템 접근 권한을 부여합니다.

- **`SignupScreen.tsx`**
  - **역할**: 신규 사원 등록(회원가입) 페이지
  - **기능**: 사전에 허가된 사원증 번호(White-list) 기반으로 계정을 생성하며, 비밀번호 재확인 및 자릿수 등 입력 폼 유효성 검사를 수행합니다.
